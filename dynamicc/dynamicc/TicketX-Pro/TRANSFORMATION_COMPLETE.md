## 💡 CUSTOMIZATION GUIDE

### Change Primary Color:
Edit `src/css/variables.css`:
```css
--primary: #6366f1;        /* Change Indigo to your color */
--primary-dark: #4f46e5;
--primary-light: #818cf8;
```

### Add More Telugu Movies:
Edit `src/js/api/movies-api.js`:
```javascript
{
    id: 17,
    title: "Movie Name",
    language: "Telugu",
    rating: 8.5,
    // ... other fields
}
```

### Modify Help Content:
Edit `pages/help.html`:
```html
<div class="faq-item">
    <div class="faq-question" onclick="toggleFAQ(this)">
        <span>Your question here</span>
    </div>
    <div class="faq-answer">Your answer here</div>
</div>
```
