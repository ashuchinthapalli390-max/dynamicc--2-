## 📝 Complete User Flow

### Step 1: Login
1. Click "Login" button in navbar
2. Pre-filled: `demo@ticketx.com` / `Demo123!`
3. Click "Login Now"
4. ✅ See "Login successful!" message

### Step 2: Browse
1. View different categories in hero section
2. Scroll through movie/train/flight/bus/event cards
3. Each card shows: Name, Rating, Price

### Step 3: Book a Ticket
1. Click "Book Now" on any card
2. Booking modal opens showing:
   - Item name with emoji
   - Price per ticket
3. Enter number of tickets (default: 1)
4. Click "Add to Cart"
5. Choose "Yes" to checkout

### Step 4: Checkout
1. Review order summary
2. See total price
3. Fill payment details (dummy - use provided values)
4. Click "Pay Now"

### Step 5: Payment Processing
1. Processing takes 2 seconds (simulated)
2. 90% chance of success
3. Success: 
   - ✅ "Payment successful!" message
   - Wallet balance updated
   - Booking confirmed with ID
   - QR code generated

### Step 6: Booking Confirmation
- Alert shows: Booking ID, Total, Status
- QR code ready for download
- New wallet balance displayed
