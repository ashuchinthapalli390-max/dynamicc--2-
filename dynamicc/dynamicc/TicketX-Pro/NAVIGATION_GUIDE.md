## 🎓 HOW TO USE TICKETX PRO

### Step 1: Create Account (2 minutes)
```
1. Go to http://localhost:3000
2. Click "Login" in navbar
3. Click "Register" link
4. Fill in:
   - Email
   - Password
   - Full Name (optional)
   - Phone (optional)
5. Click "Register"
6. You're logged in!
```

### Step 2: Browse & Book (5 minutes)
```
1. Choose category:
   - 🎬 Movies
   - 🚂 Trains
   - ✈️ Flights
   - 🚌 Buses
   - 🎪 Events

2. Fill search criteria
3. Browse results
4. Click "Book Now" on any item
5. Select seats
6. Apply promo code (optional)
7. Proceed to checkout
8. View your ticket with QR code!
```

### Step 3: Manage Bookings (Anytime)
```
1. Click "Profile" in navbar
2. Go to Dashboard
3. View your bookings
4. Download tickets
5. View QR codes
6. Cancel if needed
```

## 🛠️ TROUBLESHOOTING

### Website Won't Load?
**Solution:**
1. Check URL: http://localhost:3000
2. Refresh page: Ctrl+F5
3. Clear cache: Ctrl+Shift+Delete
4. Try different browser
5. Check server is running

### Can't Login?
**Solution:**
1. Use test account: test@ticketx.com
2. Password: Test123!
3. Or register new account
4. Check DevTools (F12) for errors

### Bookings Not Showing?
**Solution:**
1. Refresh page: Ctrl+F5
2. Check browser storage
3. Try private window
4. Create new booking

### QR Code Issues?
**Solution:**
1. Refresh page
2. Try different browser
3. Check console (F12)
4. Download instead of view

### Payment Not Working?
**Solution:**
1. Install Node.js for backend
2. Configure .env file
3. Add Razorpay keys
4. Restart server
