   ```
   Cart → Checkout → Select Payment Method → Enter Details → Pay Now
   ```

3. **Each Method Shows:**
   - Method icon
   - Appropriate input fields
   - Helper text
   - Pre-filled demo data (for testing)

## 🤖 GEMINI AI INTEGRATION

**Advanced AI Assistant powered by Google Gemini!**

### **How to Enable AI:**

#### **Step 1: Get Your API Key (FREE)**
1. Go to: https://makersuite.google.com/app/apikey
2. Click "Create API Key"
3. Copy the key

#### **Step 2: Add to TicketX Pro**
1. Click chat bubble (bottom-right)
2. Click settings icon (🔑) in chat header
3. Paste your API key
4. Done! AI is now enabled

### **What AI Can Do:**

The Gemini AI assistant understands:
- ✅ Ticket booking questions
- ✅ Seat selection guidance
- ✅ Payment methods explanation
- ✅ QR code information
- ✅ Pricing & cancellations
- ✅ Refund policies
- ✅ General TicketX Pro inquiries
- ✅ Custom follow-up questions

### **Example Conversations:**

```
USER: How do I book a ticket?
AI: You can follow these steps...
   [Full detailed explanation]

USER: What are the payment options?
AI: We offer 5 payment methods...
   [Details about each method]

USER: Can I cancel my booking?
AI: Yes! Cancellations are processed...
   [Full refund information]
```

### **Without API Key:**

If you don't add an API key, the chat still works with:
- ✅ Default responses for common questions
- ✅ Keyword-based matching
- ✅ Helpful fallback messages
- ✅ Full functionality (just not AI-powered)

### **Fallback Responses Include:**

- Pricing information
- Booking instructions
- Payment methods overview
- Seat type descriptions
- QR code details
- Refund policies
