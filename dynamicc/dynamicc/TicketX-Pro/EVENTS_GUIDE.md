## 🎯 How to Use Events

### Browse Events
1. Click "🎤 Events" in navbar
2. Or scroll to "🎪 Events, Shows & Halls" section on home
3. Click "View All Events →" button

### Filter Events
- **By Location:** Select city from dropdown
- **By Category:** Choose event type (Dance, Singing, etc.)
- **By Price:** Select price range (₹0-500, ₹500-2000, etc.)
- **Search:** Type event name, category, or keyword

### Book an Event
1. Browse or search for events
2. Click "Book Now" button
3. Login required (auto-redirects if not logged in)
4. Proceeds to seat selection
5. Complete checkout

### Event Information
Each event shows:
- Event name & category
- Location with emoji
- Date & time
- Venue details
- Star rating (out of 5)
- Available seats
- Price per ticket
- "Book Now" button

## ❓ FAQ

**Q: How do I book an event?**
A: Browse events, select one, click "Book Now", login if needed, select seats, checkout.

**Q: Can I change my password?**
A: Yes, go to Dashboard → Security tab → Change Password section.

**Q: How do I save a location?**
A: Go to Dashboard → Locations tab → Add New Location → Select city → Click Add.

**Q: What if I forget my password?**
A: Currently, contact support. Future: Reset link via email.

**Q: Can I book events without login?**
A: No, login is required. Auto-redirects to login page if needed.

**Q: Are locations saved permanently?**
A: Yes, saved in your browser's LocalStorage. Clear browser data to remove.

**Q: Can I logout from all devices?**
A: Yes, go to Security tab and click "Logout All Devices" button.

**Q: What payment methods are supported?**
A: Currently dummy data. Future versions will support multiple payment gateways.
