🎯 HOW TO USE THE NEW FEATURES
─────────────────────────────────────────────────────────────────────────────

BROWSE EVENTS:
1. Click "🎤 Events" in navbar
2. OR scroll to "Events, Shows & Halls" on home
3. OR visit pages/events.html

FILTER EVENTS:
1. Select location from dropdown
2. Select category from dropdown
3. Select price range
4. Results update in real-time

SEARCH EVENTS:
1. Type in search box
2. Results appear instantly
3. Filter results further

BOOK EVENT:
1. Click "Book Now" button
2. Login if needed
3. Proceed to checkout

MANAGE LOCATIONS:
1. Go to Dashboard
2. Click "📍 Locations" tab
3. Click "Add New Location"
4. Select city and add notes
5. Click "Add Location"
6. View or remove locations

CHANGE PASSWORD:
1. Go to Dashboard
2. Click "🔐 Security" tab
3. Fill change password form
4. Click "Change Password"
5. Success notification
6. Re-login required next time

LOGOUT ALL DEVICES:
1. Go to Dashboard
2. Click "🔐 Security" tab
3. Scroll to "Danger Zone"
4. Click "Logout All Devices"
5. Confirm on dialog
